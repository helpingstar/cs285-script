# CS285-Script

* [**Course Page**](https://rail.eecs.berkeley.edu/deeprlcourse/)
* [**Video Page**](https://www.youtube.com/playlist?list=PL_iWQOsE6TfVYGEGiAOMaOzzv41Jfm_Ps)

This is an unofficial script for CS285.

The script was generated using Whisper and then manually edited while watching the lectures.

Equations are formatted using LaTeX notation for readability, but not for rendering purposes. Efforts were made to make the text as readable as possible, and some words are condensed into mathematical symbols. However, not all words are condensed. 
* expectation -> E
* the log(arithm) of ... -> log{..}
* probability of ... -> p(...)

The script may be continually revised as long as it does not significantly impede the understanding of the lecture content.
